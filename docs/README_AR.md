# تطبيق صيادلة العرب في ألمانيا

تطبيق ثلاثي اللغة (الإنجليزية · الألمانية · العربية) لمصطلحات الصيدلة والطب،
ومحادثات الصيدلية الواقعية، وأسئلة الاستعداد للاختبارات (FSP / Kenntnisprüfung)
لصيادلة العرب الذين يعملون أو يتدربون في ألمانيا.

> ⚠️ **تنبيه قانوني.** المحتوى لأغراض تعليمية فقط ولا يُعد استشارة طبية. يجب
> دائماً التحقق من الجرعات والاستطبابات في المصادر الرسمية الحديثة
> (Fachinformation, ABDA, Rote Liste, Lauer-Taxe) ومراجعة مختص قبل الاستخدام
> السريري.

---

## بنية المستودع

```
.
├── app/        # تطبيق Flutter للجوال (Android / iOS / ويب)
├── data/       # ملفات JSON: مصطلحات، محادثات، أدوية، اختبارات
├── backend/    # مخطط PostgreSQL
├── scripts/    # سكربت تعبئة قاعدة البيانات (Node.js)
└── docs/       # الخطط والوثائق العربية
```

## محتوى قاعدة البيانات

| الملف             | عدد العناصر | الوصف                                                     |
| ----------------- | ----------- | --------------------------------------------------------- |
| `categories.json` | 10          | تصنيفات المصطلحات (مرض، عَرَض، صنف دوائي، …).             |
| `terms.json`      | 300         | مصطلحات طبية وصيدلية ثلاثية اللغة مع تعريفات وأمثلة.      |
| `drugs.json`      | 30          | أدوية مع المادة الفعالة والصنف والاستطباب والجرعة والتحذيرات. |
| `dialogues.json`  | 30          | محادثات صيدلية واقعية مع ترجمة كاملة للغات الثلاث.         |
| `quizzes.json`    | 12          | أسئلة اختيار من متعدد مع شرح الإجابة.                      |

البيانات الحالية هي **بذرة عالية الجودة** قابلة للتوسعة، وليست العشرة آلاف
مصطلح المذكورة في الخطة طويلة الأمد. ترجمة المصطلحات الطبية تحتاج إلى مراجعة
صيدلي ومتخصص، لذلك نُفضّل النوعية على الكمية. خطة التوسعة في
[`docs/plan.md`](plan.md).

## شاشات التطبيق

1. **المصطلحات** — تصفح 300 مصطلح مع تصفية حسب التصنيف وبحث متعدد اللغات.
2. **المحادثات** — 30 سيناريو حقيقياً (وصفة، استشارة، تأمين، BtM، …) مع
   إمكانية التبديل بين اللغات بضغطة واحدة.
3. **الأدوية** — قائمة الأدوية مع المادة الفعالة، الاستطباب، الجرعة، الفلترة بين
   Rx (بوصفة) و OTC (بدون وصفة)، والتحذيرات.
4. **الاختبار** — أسئلة عشوائية متعددة الخيارات مع شرح الإجابة.
5. **البحث** — بحث شامل عبر المصطلحات والأدوية بالإنجليزية أو الألمانية أو
   العربية.

## التشغيل المحلي

```bash
cd app
flutter pub get
flutter run            # على الجهاز / المحاكي
flutter run -d chrome  # نسخة الويب
flutter test           # اختبارات تحقق من البيانات
flutter analyze        # فحص الجودة
```

## قاعدة البيانات

ملف المخطط [`backend/schema.sql`](../backend/schema.sql) يُنشئ جداول PostgreSQL
للتصنيفات، المصطلحات، الأدوية، المحادثات، الاختبارات، وتقدم المستخدم
(صناديق Leitner للتكرار المتباعد).

```bash
psql "$PGURL" -f backend/schema.sql

cd scripts
npm install pg
PGURL="postgres://user:pass@host:5432/db" node seed.js
```
